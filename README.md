Clone 專案 並進入資料夾。
--------------------------------------------------------------------
複製環境變數env.example 改成 .env (並填入自己的資料庫資訊)。

安裝依賴：uv sync 

macOS電腦如果執行./dev.sh不允許
請先使用：chmod +x dev.sh
-------------------------------------------------------------------------
開發 API：請去 routes/ 找對應自己的檔案，不要動 models.py，請在裡面實現 CRUD 功能。

測試：打開 http://127.0.0.1:8000/docs 確認 API 能動。

API 測試：請確保在 /docs 測試成功後再進行 git commit。
